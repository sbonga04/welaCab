# Welacab

A Whatsapp based ride hailing app built for small towns and rural areas undeserved by uber or bolt.
this system is built with Spring Boot, mySQL and Twillio

## How it works
- Riders and drivers interact entirely on WhatsApp -> no app download needed.
- Riders request riders by providing pickup and dropoff locations.
- Available drivers get notified instantly via WhatsApp
- Built for areas like umhlabuyalingana, Kwazulu-Natal where this technology doesn't exist
